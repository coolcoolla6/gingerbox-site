Privacy Policy

Last updated: June 2026

Overview

GingerBox is a recipe and ingredient management application. Most data is stored locally on the user's device.

Information We Collect

GingerBox does not require account registration.

If you choose to enable analytics, anonymous usage information may be collected through Firebase Analytics.

To improve app stability, diagnostic crash information may be collected through Firebase Crashlytics.

Google Drive Backup

GingerBox provides optional backup and restore features using Google Drive. Data is uploaded only when the user explicitly initiates a backup.

Data Storage

Recipe data, ingredient data, shopping lists, and application settings are stored locally on the user's device unless the user chooses to back them up to Google Drive.

Third-Party Services

  • Google Firebase Analytics
  • Google Firebase Crashlytics
  • Google Drive API

Contact

For privacy-related questions, please contact: support@gingerbox.app
